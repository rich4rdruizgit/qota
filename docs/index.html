Features

Everything you need.
Nothing you don't.

A single Bash script. No server. No account. No background daemon.

📊

Live token counter

Tracks input, output, cache read and cache write tokens in real time, per session.

🔔

Smart notifications

Native macOS alerts at every 10% of usage. Quiet info early, critical warnings near the limit.

🧠

Auto plan detection

Detects your Claude plan automatically by comparing your historical peak usage.

⚡

Zero overhead

~100 lines of Bash. Runs every 5 minutes. Uses no CPU between refreshes.

Privacy

Your conversations
stay yours.

Qota only reads the numeric fields from Claude Code's local JSONL files. It never touches message content.

✓ Reads only input_tokens, output_tokens, cache_read_input_tokens, cache_creation_input_tokens and the model name.
✓ Everything runs locally on your Mac. No internet connection required.
✓ No telemetry, no analytics, no external tracking of any kind.
✓ The full source is ~100 lines of pure Bash — auditable in minutes.

Installation

Up and running
in 3 steps.

Requires macOS 12+ and SwiftBar.

1

Install SwiftBar

brew install --cask swiftbar

2

Clone this repo

git clone https://github.com/your-user/qota

3

Run the installer

chmod +x install.sh && ./install.sh
